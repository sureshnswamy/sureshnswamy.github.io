Javascript Basics

How does Javascript compare to HTML and CSS?

All three languages are used to create a website to do different functions in it. HTML is used to define the basic structure and the actual content of a page. Then CSS is used to design the page like how everything in the page looks. Javascript (JS) makes a page livelier, or make it user responsive. JS is a programming language which runs in your browser. JS you can perform calculations, logical conditions and all programming functionality. HTML alone cannot do it.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'

Control flow statements are those that make decisions about what happens in a program. A programmer needs to take care that the conditions used do not contradict one another. A control flow statement modifies a program's control flow. Statements like if...else, switch...case, for, for...in, while and do...while A control structure additional contains another statement which is executed under specified conditions, by modification and/or validation of the environment. Furthermore, loops are structures which repeat their statements while the environment is validated by a given test, or "condition." Explain the difference between accessing data from arrays and object literals?

In Javasscript you can create both Object and Array literals on the fly. The syntax for Object literal is:

Enclose it in curly brackets
separate properties with comma
separate keys and values with a colon

similalry for array literal :

Enclose it in square brackets
separate each element with comma

A key difference between Array and Objects is Arrays are really good at storing ordered lists and ordering things while removing or splicing elements takes more processing time. Objects are great at representing schemas and augmenting or deleting properties is easy, however, they are not good for preserving order.

Explain what functions are and why they are useful

Functions are the backbone of Javascript. Functions can be used as a tool to structure large programs, reduce repetition, to associate names with subprograms, and to isolate subprograms from each other. A function is created with keyword function have a set of parameters and a body wich contains the statements to be executed when the function is called. For example:

var add = function(y) {
return y+y;
}
